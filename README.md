# Homebrew Tap

Homebrew tap for tasselx apps.

## Installation

```bash
brew tap tasselx/tap
```

## Available Apps

### Keyden

macOS menu bar TOTP authenticator.

```bash
brew install --cask keyden
```

**Requirements:** macOS 12.0+ (Monterey)

[GitHub Repository](https://github.com/tasselx/Keyden)
